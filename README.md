# 抖音评论关键词私信工具

> 面向**已授权账号**的小批量评论筛选与私信辅助工具。

本工具运行在本地，提供一个浏览器图形界面：按搜索词找到候选视频，扫描视频评论，按关键词筛选出命中评论的用户，并向其发送一条固定内容的私信。整个过程使用真实浏览器（Playwright 驱动 Chrome / Edge），登录与验证由人工完成。

## 功能特性

- **关键词筛选**：按搜索词检索视频，按评论关键词命中筛选目标用户。
- **预览 / 发送两种模式**：预览模式只筛选、不打开私信窗口、不写入历史，便于先核对再发送。
- **人工登录优先**：遇到登录或验证码时进入「等待登录」状态交由人工处理，**不识别、不绕过验证码**。
- **频率控制**：可配置最小/最大发送间隔、候选视频上限、每视频评论上限、滚动轮数等，避免高频操作。
- **去重历史**：记录已联系用户，避免重复打扰。
- **结果导出**：运行结果可导出为 CSV。
- **单文件打包**：可打包为免安装的 Windows 可执行文件（仍需本机安装 Chrome / Edge）。

## 运行环境

- Node.js 20+（源码运行）
- Google Chrome 或 Microsoft Edge（浏览器自动化所需）
- 操作系统：Windows（打包目标）；源码亦可在其他平台运行

## 快速开始

```bash
npm install
npm start
```

启动后在浏览器打开终端提示的本地地址（默认本机端口），按界面填写任务参数即可。建议先用**预览模式**确认筛选结果，再切换到发送模式。

## 打包为单文件 exe

```bash
npm run build:exe
```

产物为 `dist/DouyinKeywordCommentDM-v<version>.exe`，详见 [docs/packaging.md](docs/packaging.md)。

## 数据存储

源码运行时数据保存在仓库 `data/` 目录；打包版保存在 `%LOCALAPPDATA%\DouyinKeywordCommentDM\data`。其中包含**登录态、运行历史、截图**等敏感信息，已通过 `.gitignore` 排除，请勿提交或分享。

## 目录结构

```
server.mjs        本地 HTTP 服务与任务调度
public/           Web 图形界面
lib/              核心逻辑：页面操作、任务运行、存储
scripts/          单文件打包脚本
packaging/        打包引导
docs/             打包说明与平台探索记录
test/             单元测试（node --test）
```
